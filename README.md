# ALTINS1_Analizi

gr altın, ons altın, $/TL, faiz, enflasyon ve gündem değerlerini baz alarak güncel ALTINS1 fiyatının analizini yapar ve al/sat sinyali üretir. Gümüş ve petrol için de eklentiler yapılacak.

---

## 🚀 PROGRAMI BAŞLATMA

### Windows

**`start.bat`** dosyasına çift tıklayın.

### Manuel Başlatma (Terminal)

```bash
# 1. Virtual environment'ı aktif edin
.venv\Scripts\activate

# 2. Uygulamayı başlatın
streamlit run main.py
```

> **Not:** İlk çalıştırmadan önce gerekli paketlerin yüklü olduğundan emin olun:
> ```bash
> pip install -r requirements.txt
> ```
